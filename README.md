### Install

#### Clone this repo

```
git clone https://github.com/storbeck/cassandra-client.git
```

#### Install dependencies

```
npm install
```

### Usage

#### Run the app

```
npm run start
```

#### Build the app

```
npm run build
```


#### Package the app

Binaries will be found in ./release/

```
npm run package
```

#### Test the app (after `npm run build` || `yarn run build`)
```
npm run prod
```
